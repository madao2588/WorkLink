# WorkLink

这是 WorkLink 项目的仓库，包含后端（FastAPI）与前端（Flutter）代码。

## 项目结构（摘要）

- `backend/`：FastAPI 后端服务，使用 DuckDB 本地开发数据库，生产可切换到 PostgreSQL。
- `lib/`：Flutter 应用源码。

## 前端说明

该 Flutter 项目基于 `my_first_app` 启动模板构建，包含常用的入门资源。

如果你只关心 Flutter 模块，这里是一些快速参考：

- Learn Flutter: https://docs.flutter.dev/get-started/learn-flutter
- Write your first Flutter app: https://docs.flutter.dev/get-started/codelab

更多细节请查看 `backend/README.md` 获取后端运行与联调说明。
